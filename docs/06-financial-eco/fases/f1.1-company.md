# F1.1 — Company (Sociedad)

> **Fase:** F1.1 · **Esfuerzo:** Bajo · **Sesión:** 2/N

---

## 0. Objetivo

Crear el primer dominio financiero del ecosistema S/4HANA: **Company (Sociedad)**,
expuesto como `/odata/finance/company-odata`. La sociedad es la entidad raíz en SAP:
toda factura, cliente, proveedor, cuenta y pago pertenece a una sociedad.

---

## 1. Modelo de datos

| Campo | Tipo | PK | Descripción |
|-------|------|----|-------------|
| `id` | `STRING` | ✅ | Código de sociedad (p.ej. `"1000"`, `"2000"`) |
| `nombre` | `STRING` | | Razón social |
| `moneda` | `STRING` | | Moneda local (p.ej. `"USD"`, `"EUR"`) |
| `pais` | `STRING` | | País (p.ej. `"US"`, `"DE"`) |
| `createdAt` | `DATE` | | Timestamp (Sequelize) |
| `updatedAt` | `DATE` | | Timestamp (Sequelize) |

- `id` es `STRING` PK (código SAP, no auto-incremental).
- Tabla física: `companies`.
