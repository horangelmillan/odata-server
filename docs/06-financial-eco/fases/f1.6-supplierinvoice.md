# F1.6 — Dominio `supplierinvoice` (Factura proveedor / MM)

> **Sub-fase:** F1.6 · **Esfuerzo:** Medio · **Patrón:** `docs/02-patrones/05-odata-module-pattern.md`
> **Depende de:** F1.1, F1.3.

## 0. Objetivo
Crear `core/supplierinvoice/` — factura de proveedor (MM), análoga a `invoice` pero con `supplier`.

## 1. Modelo `core/supplierinvoice/model/supplierinvoice.odata.model.ts`
```typescript
@Table({ tableName: "supplierinvoices" })
export class SupplierInvoiceOData extends Model<SupplierInvoiceOData> {
    @Column({ dataType: DataTypes.STRING, isPrimaryKey: true }) supplierInvoiceId!: string; // "SI0001"
    @Column({ dataType: DataTypes.STRING }) docNumber!: string;                            // "5100000001"
    @Column({ dataType: DataTypes.STRING }) company!: string;                             // "1000"
    @Column({ dataType: DataTypes.STRING }) supplierId!: string;                          // "S00001"
    @Column({ dataType: DataTypes.DATE }) date!: Date;
    @Column({ dataType: DataTypes.DATE }) dueDate!: Date;
    @Column({ dataType: DataTypes.DECIMAL }) netAmount!: number;
    @Column({ dataType: DataTypes.DECIMAL }) taxAmount!: number;
    @Column({ dataType: DataTypes.DECIMAL }) grossAmount!: number;
    @Column({ dataType: DataTypes.STRING }) status!: string;                             // PENDIENTE | PAGADA | VENCIDA
    @Column({ dataType: DataTypes.STRING }) currency!: string;                           // "EUR"
    @Column({ dataType: DataTypes.DATE }) createdAt!: Date;
    @Column({ dataType: DataTypes.DATE }) updatedAt!: Date;

    @BelongsTo(() => SupplierOData, { relation: [{ foreignKey: "supplierId", targetKey: "supplierId" }] })
    supplier!: SupplierOData;

    @BelongsTo(() => CompanyOData, { relation: [{ foreignKey: "company", targetKey: "code" }] })
    companyRef!: CompanyOData;
}
```

## 2. Resto del módulo
- `interface/supplierinvoice.interface.ts` → `ISupplierInvoice`.
- `dto/supplierinvoice.dto.ts` → `SupplierInvoiceCreateDTO` / `SupplierInvoiceUpdateDTO`.
- `controller/supplierinvoice.odata.controller.ts` → `ODataControler` (`allowedMethod: ["get"]`).
- `service/supplierinvoice.service.ts` → singleton.
- `main.ts` → barrel.

## 3. Registro
- `datasource.ts`: añadir `SupplierInvoiceOData`.
- `odata.service.ts`: añadir `new SupplierInvoiceODataController()`.
