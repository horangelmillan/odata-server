# F1.7 — Dominio `invoiceitem` (Línea de factura)

> **Sub-fase:** F1.7 · **Esfuerzo:** Medio · **Patrón:** `docs/02-patrones/05-odata-module-pattern.md`
> **Depende de:** F1.4, F1.5.

## 0. Objetivo
Crear `core/invoiceitem/` — posición/línea de factura, enlazada a `invoice` y `glaccount`.
Cierra las navegaciones `@HasMany` declaradas en F1.4 y F1.5.

## 1. Modelo `core/invoiceitem/model/invoiceitem.odata.model.ts`
```typescript
@Table({ tableName: "invoiceitems" })
export class InvoiceItemOData extends Model<InvoiceItemOData> {
    @Column({ dataType: DataTypes.STRING, isPrimaryKey: true }) itemId!: string;    // "II00001"
    @Column({ dataType: DataTypes.STRING }) invoiceId!: string;                     // "I00001"
    @Column({ dataType: DataTypes.STRING }) glAccount!: string;                     // "0000123400"
    @Column({ dataType: DataTypes.STRING }) material!: string;                      // "MAT-001"
    @Column({ dataType: DataTypes.INTEGER }) quantity!: number;
    @Column({ dataType: DataTypes.DECIMAL }) netAmount!: number;
    @Column({ dataType: DataTypes.DECIMAL }) taxAmount!: number;
    @Column({ dataType: DataTypes.DATE }) createdAt!: Date;
    @Column({ dataType: DataTypes.DATE }) updatedAt!: Date;

    @BelongsTo(() => InvoiceOData, { relation: [{ foreignKey: "invoiceId", targetKey: "invoiceId" }] })
    invoice!: InvoiceOData;

    @BelongsTo(() => GlAccountOData, { relation: [{ foreignKey: "glAccount", targetKey: "glAccount" }] })
    glAccountRef!: GlAccountOData;
}
```

## 2. Resto del módulo
- `interface/invoiceitem.interface.ts` → `IInvoiceItem`.
- `dto/invoiceitem.dto.ts` → `InvoiceItemCreateDTO` / `InvoiceItemUpdateDTO`.
- `controller/invoiceitem.odata.controller.ts` → `ODataControler` (`allowedMethod: ["get"]`).
- `service/invoiceitem.service.ts` → singleton.
- `main.ts` → barrel.

## 3. Registro
- `datasource.ts`: añadir `InvoiceItemOData`.
- `odata.service.ts`: añadir `new InvoiceItemODataController()`.

## 4. Criterios de aceptación
- [ ] `core/invoiceitem/` completo.
- [ ] Navegaciones `invoice → items → glAccount` resoluble tras F3.
- [ ] `pnpm test` en verde.
