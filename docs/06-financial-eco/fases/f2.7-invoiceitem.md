# F2.7 — Seed `invoiceitem`

> **Sub-fase:** F2.7 · **Esfuerzo:** Medio · **Depende de:** F2.4, F2.5.

## 0. Objetivo
`scripts/seed/data/invoiceitems.json` con ~1-4 líneas por factura (`II00001`…), enlazadas a
`invoiceId` y `glAccount`. La suma de `netAmount`/`taxAmount` de las líneas ≡ total de la factura.

```json
[
  { "itemId": "II00001", "invoiceId": "I00001", "glAccount": "0000123400", "material": "MAT-001",
    "quantity": 2, "netAmount": 1000.00, "taxAmount": 210.00,
    "createdAt": "2026-02-10T00:00:00.000Z", "updatedAt": "2026-02-10T00:00:00.000Z" }
]
```

## 1. Criterios de aceptación
- [ ] Líneas cubren todas las facturas; enlazan a glaccounts válidos.
- [ ] Suma de líneas ≡ total de factura.
- [ ] `pnpm seed` reproduce idéntico.
