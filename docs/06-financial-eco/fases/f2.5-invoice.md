# F2.5 — Seed `invoice`

> **Sub-fase:** F2.5 · **Esfuerzo:** Medio · **Depende de:** F2.2, F2.4.

## 0. Objetivo
`scripts/seed/data/invoices.json` con ~50 facturas cliente (`I00001`…). Coherentes: importes,
fechas repartidas en 6 meses, estado inicial (se recalcula en F3).

Reglas de coherencia:
- `grossAmount = netAmount + taxAmount`.
- IVA 21% (o 10%/0% en algunas).
- `dueDate = date + 30d` (paymentTerms 30D) o 60d.
- ~60% tendrán pago en F2.8 (PAGADA), ~25% PENDIENTE, ~15% VENCIDA.

```json
[
  { "invoiceId": "I00001", "docNumber": "1900000001", "company": "1000", "customerId": "C00001",
    "date": "2026-02-10", "dueDate": "2026-03-12", "netAmount": 1000.00, "taxAmount": 210.00,
    "grossAmount": 1210.00, "status": "PENDIENTE", "currency": "EUR",
    "createdAt": "2026-02-10T00:00:00.000Z", "updatedAt": "2026-02-10T00:00:00.000Z" }
]
```

## 1. Criterios de aceptación
- [ ] ~50 facturas con IDs fijos; importes coherentes (net+tax=gross).
- [ ] `pnpm seed` reproduce idéntico.
